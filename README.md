🌐 Flask Web Projects Collection

  

A collection of Flask-based web applications built to demonstrate backend development, API integration, and full-stack Python web skills.

Each project is designed as a standalone mini web app, focusing on real-world concepts like CRUD systems, APIs, dashboards, and interactive UI design.


---

📦 Projects Included

🌦️ Weather Web App

A live weather application that fetches real-time weather data using an external API.

Features:

City search system

Live weather data display

Temperature & wind tracking

API integration



---

🤖 Flask Chatbot Web App

A simple rule-based chatbot built with Flask.

Features:

Interactive chat UI

Predefined AI-style responses

Keyword-based logic system

Dynamic conversation rendering



---

📝 Todo Web App

A task management system built with CRUD functionality.

Features:

Add tasks

Delete tasks

Real-time list updates

In-memory data handling



---

⚔️ RPG Dashboard Web App

A game-inspired analytics dashboard for RPG character stats.

Features:

Character stat tables

PowerScore ranking system

Class-based comparison

Leaderboard visualization



---

🌐 Personal Portfolio Web App

A multi-page developer portfolio built with Flask routing.

Features:

About page

Projects showcase

Skills section

Contact form UI

Navigation system
